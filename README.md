# Coffee-machine-project
A command-line based Coffee Machine Simulator built using Python.
The program mimics the behavior of a real coffee vending machine, handling user orders, processing coins, managing resources, and generating reports.

This project focuses on logic building, state management, and structured programming, making it a strong beginner-to-intermediate level project.
🚀 Features<br>
☕ Multiple drink options:<br>
Espresso<br>
Latte<br>
Cappuccino<br><br>
💰 Coin-based payment system:<br>
Accepts quarters, dimes, nickels, pennies<br>
Calculates total and returns change<br><br>
📦 Resource management:<br>
Tracks water, milk, and coffee<br>
Prevents orders if resources are insufficient<br><br>
📊 Reporting system:<br>
Displays current resource levels and profit<br>
🔁 Continuous operation until turned off<br><br>
🕹️ How It Works<br>
User selects a drink (espresso, latte, cappuccino)<br>
Machine checks if resources are sufficient<br>
User inserts coins<br><br>
Machine:<br>
Validates payment<br>
Returns change (if any)<br>
Updates resources<br>
Coffee is “served” ☕<br><br>
Special commands:<br>
report → shows resources<br>
off → turns off machine<br><br>
🛠️ Tech Stack<br>
Language: Python<br>
Concepts Used:<br>
Dictionaries for menu & resources<br>
Functions for modular logic<br>
Conditional statements<br>
Loops for continuous operation<br>
